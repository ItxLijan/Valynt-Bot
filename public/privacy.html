🔒 Datenschutzerklärung

Zuletzt aktualisiert: April 2026

1. Verantwortlicher

Dieser Discord-Bot wird betrieben von Valynt. Bei Fragen zum Datenschutz wende dich bitte an den Server-Administrator.

2. Welche Daten werden gespeichert?

Der Bot speichert folgende Daten in einer MariaDB-Datenbank:

  • Discord User-ID und Server-ID (zur Identifizierung)
  • XP-Punkte, Level und Coins (für das Progression-System)
  • Geburtstag (nur wenn vom Nutzer eingetragen, Format MM-DD)
  • Minecraft-Username und -Version (nur wenn vom Nutzer eingetragen)
  • Daily-Streak-Daten (Datum der letzten Abholung, Streak-Anzahl)
  • Shop-Käufe (User-ID, gekauftes Item, Lieferstatus)

3. Zweck der Datenspeicherung

Die Daten werden ausschließlich für den Betrieb des Discord-Bots verwendet:

  • Verwaltung von Rangsystemen und Leaderboards
  • Verarbeitung von Shop-Käufen
  • Geburtstagserinnerungen
  • Tägliche Belohnungen (Streak-System)

4. Datenweitergabe

Deine Daten werden nicht an Dritte weitergegeben. Alle Daten verbleiben auf dem Server des Bot-Betreibers.

5. Datenlöschung

Du kannst jederzeit die Löschung deiner Daten beantragen. Wende dich dafür an einen Server-Administrator. Wenn du den Server verlässt, werden deine Daten nicht automatisch gelöscht.

6. Dashboard-Login

Für das Web-Dashboard wird Discord OAuth2 genutzt. Dabei werden folgende Daten temporär gespeichert:

  • Discord User-ID und Username (für die Anzeige)
  • Avatar-URL
  • Liste der verwalteten Server

Diese Session-Daten werden nach 7 Tagen automatisch gelöscht.

7. Cookies

Das Dashboard verwendet ein Session-Cookie (connect.sid) um die Anmeldung zu speichern. Dieses Cookie ist technisch notwendig für den Betrieb des Dashboards und enthält keine persönlichen Daten.
